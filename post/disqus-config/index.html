GreyShade主题下disqus的配置

disqus账号注册

要想使用disqus作为自己的评论系统，首先需要一个disqus账号，可以上disqus注册一个，最后需要邮箱激活，因此注册时请提供能够使用的邮箱，我使用的是Gmail邮箱，注册的步骤就不细说了。

登记你的站点

这一步比较重要，如果没有这一步的话，就无法得到下面你使用的disqus_shortname了，我在刚开始就是不知道这一步，将我的disqus用户名配置成disqus_shortname，导致出现

We were unable to load Disqus

的错误,在这个错误里给出一个trouble shooting的链接，我就是按照这个链接的提示找到问题所在的。

登记站点很简单，sing in到disqus后从http://disqus.com/admin就可以登记你的站点，如果以前没有添加过站点，那么第一次打开的是这个界面

注意上面的disqus url前面需要填写的部分就是你站点的disqus_shortname，将会在下面用到。

如果以前登记过站点，这次只需要Add new site就行了

Hugo博客生成系统配置

以下的配置都是在greyshade主题下的配置步骤，不同主题可能配置方法会稍有不同

  • 在需要开放评论的页面中加入comments = true

    +++
Categories = ["Hugo"]
Description = "Hugo博客相关"
Tags = ["Hugo"]
date = "2015-07-19T20:04:30+09:00"
menu = "main"
title = "利用Hugo搭建博客系统"
comments = true
+++

  • 然后配置config.toml，在params中加入下面配置

    disqus_short_name = "上面在disqus登记站点时的disqus_shortname"

博客网页更新

做好以上的配置之后，可以在本地利用hugo server -w进行预览，如果没有问题了就可以重新生成博客静态网页，并更新到站点上去。如果使用的是github，更新步骤可以参照我上一篇博客利用Hugo搭建博客系统

至此，disgus配置的步骤也就完成了。
